# Top Down Shooter

## Map Editor

### Key Bindings

| Key                 | Action                                                  |
|---------------------|---------------------------------------------------------|
| **Q**               | Quit                                                    |
| **S**               | Save                                                    |
| **O**               | Open                                                    |
| **E**               | Export                                                  |
| **_Shift_ + H**     | Hide/Show Tile Properties                               |
| **T**               | Select Tileset                                          |
| **F**               | Fill Layer                                              |
| **R**               | Rotate Tile                                             |
| **Space**           | Place Tile                                              |
| **+/-**             | Change Tile Scale (ctrl for changing the tile size)     |
| **_Ctrl_ + H/L**    | Switch between Palette and Map                          |
| **Vim Keys (HJKL)** | to navigate in selected panel                           |
| **G**               | to go to start/end of column (with Shift for end/start) |
